What it does

Everything Your Server Needs

🔒
Moderation Suite
Ban, kick, timeout, warn, purge — all with DM notifications, reason tracking, full mod history, and auto-logging to a dedicated channel.
🎵
Music Player
Play any YouTube video or search by name directly. Full queue system with GUI controls (skip, pause, resume, stop) on every Now Playing embed. Auto-inactivity disconnect.
🎫
Ticket System
Interactive setup wizard — set category, staff role, and panel message. Auto-numbered tickets, claim system, transcripts saved on close, add/remove members.
🎉
Giveaway Manager
Timed giveaways with reaction entry — 30s to 2d durations. Auto-picks random winner when time expires. Reroll command for new winners.
🔄
Auto Emoji Swap
Detects channel lock/unlock from any bot — Dyno, Carl, MEE6, Raid Protect — and flips emoji automatically. Compatible with your existing setup.
✅
Verification System
Set unverified and member roles. Staff can verify/unverify users with a command. Keeps unverified members separated from the main server.
🛡️
NSFW Auto-Delete
Automatically detects and deletes NSFW content. Sends a temporary warning to the user. Keeps your server clean without manual effort.
📋
Full Audit Logging
Every mod action (ban, kick, timeout, warn) is logged with moderator info, reason, and timestamp. Manual action logging for off-Discord actions too.
⭐
Promotion Tracker
Set up to 15 ordered staff ranks. When someone gets Trial Mod → posts a welcome. Promotion to Mod, Senior Mod, Admin — each gets its own styled announcement.
🎮
Games & Fun
Tic-Tac-Toe, Trivia with button answers. Memes, quotes, roasts, fake Nitro, random cat/dog/fox images. Something for every server.
⚡
Access Levels
Three-tier permission system — LV1 basic, LV2 moderation, LV3 admin. Grant any role or user access without giving them Discord permissions.
📡
Auto Update Pings
Detects "Exploit Update", "Roblox Update" and similar phrases. Auto-pings the matching staff role. Keeps the server informed without manual work.

How to use

Commands

.exec ban [@user] [reason]Permanently ban a user — DMs them before banning
.exec kick [@user] [reason]Kick a user — DMs them the reason
.exec timeout [@user] [mins] [reason]Mute a user for a set duration
.exec warn [@user] [reason]Warn a user — saved to their history
.exec warns [@user]Show all warns for a user
.exec clearwarns [@user]Clear all warns for a user
.exec purge [amount]Bulk delete 1–100 messages
.exec lock [#ch] [reason]Lock a channel
.exec unlock [#ch] [reason]Unlock a channel
.exec view [#ch] [disable/enable] [@role]Hide or show a channel from a role
.exec slowmode [#ch] [seconds]Set slowmode (0 = off)
.exec unban [userId]Unban a user by ID
.exec userbansShow all banned users with reasons — paginated
.exec modhis [@user]View full moderation history for a user
.exec actionlog [@user] [action] [reason]Manually log a mod action to bot-logs
.exec play [url or song name]Play from YouTube — link or search by name
.exec skipSkip the current song
.exec pause / .exec resumePause or resume playback
.exec stopStop music and leave the voice channel
.exec queueShow the current music queue
.exec musicbot add [#vc]Move the bot to a specific voice channel
Music GUI ButtonsSkip, Pause, Resume, Stop, Queue buttons appear on every Now Playing embed
.exec setupticketRun the interactive ticket setup wizard (category, role, panel)
.exec ticket closeClose current ticket — saves transcript to bot-logs
.exec ticket add [@user]Add a user to the current ticket
.exec ticket remove [@user]Remove a user from the current ticket
.exec ticket claimClaim this ticket as your own
.exec ticket rename [name]Rename the ticket channel
.exec ticket transcriptPost a transcript of the ticket in chat
.exec ttt [@user]Play Tic-Tac-Toe against another user — GUI board
.exec triviaMultiple choice trivia question — click buttons to answer
.exec memeFetch a random meme from Reddit
.exec quotePost an inspirational quote
.exec roast [@user]Roast a user with a random insult
.exec fakenitroSend a fake Nitro gift (joke)
.exec random [cat/dog/fox/meme]Random image from chosen category
.exec afk [reason]Set AFK — bot notifies anyone who pings you
.exec giveaway [duration] [prize]Start a timed giveaway with reaction entry
.exec rerollReroll the last giveaway for a new winner
.exec poll [question] | [opt1] | [opt2]Create a multi-option poll
.exec announce [message] [#channel]Send a styled announcement to a channel
.exec setupInteractive setup dashboard — shows progress and what's missing
.exec botlogs set [#channel]Set the channel for mod action logs
.exec botstatus [#channel]Set channel for online/offline messages
.exec botupdates [#channel]Set channel for changelog posts
.exec watch add/remove/list [#ch]Manage emoji swap watched channels
.exec verify setrole [@unverified] [@member]Configure verification roles
.exec verify / unverify [@user]Verify or unverify a user
.exec promotionrole set [@r1] [@r2]...Set ordered staff ranks (Trial → Mod → Admin etc.)
.exec promotionchannel [#channel]Set channel for promotion announcements
.exec access [@role/@user] [LV1/LV2/LV3]Set permission tier for a role or user
.exec wl channel/access add/remove/listManage channel and access whitelists
.exec GlobalLogs disable/enableToggle all bot logging globally
.execShow the bot info card (overview)
.exec helpFull paginated command list with GUI navigation
.exec pingBot latency and API response time
.exec serverinfoServer stats — members, boosts, channels
.exec userinfo [@user]Full user profile — join date, roles, status
.exec botworksWhat the bot is monitoring and recent activity
.exec say [message]Post a message as the bot in the current channel
.exec info [command]Show usage info for any specific command
