What it does

Features

🔒
Auto Emoji Swap
Automatically flips channel emojis from 🟢 to 🔴 when locked by any bot — Raid Protect, Dyno, Carl, or .execVØID itself.
🛡️
Built-in Lock System
Lock and unlock channels natively with reason support. Sends a styled alert directly in the locked channel.
📋
Whitelist Management
Channel whitelist lets roles/users chat even when locked. Access whitelist grants command permissions without server roles.
🔍
Audit Log Detection
Uses Discord's audit log to reliably detect lock/unlock events from any source, with duplicate protection.
⚡
Works With Any Bot
Compatible with Raid Protect, Dyno, Carl-bot, MEE6 and more. If it changes channel permissions, .execVØID sees it.
🎨
Styled Responses
Dark purple embeds with animated emoji, reason support, developer pings, and a clean footer on every response.

How to use

Commands

.exec serverinfo Shows bot latency, API latency, server name, creator and purpose
.exec lock [#channel] [reason] Locks a channel with a reason. Sends a styled alert and pings @TiRex Developer
.exec unlock [#channel] [reason] Unlocks a channel and sends an unlock confirmation message
.exec whitelist channel @role/@user Adds a role or user to the channel whitelist — they can chat even when locked
.exec channel whitelist Lists all roles and users in the channel whitelist
.exec unwhitelist channel @role/@user Removes a role or user from the channel whitelist
.exec whitelist access @role/@user Grants a role or user access to bot commands without needing Manage Channels
.exec whitelist access member Lists all roles and users in the access whitelist
.exec unwhitelist access @role/@user Removes a role or user from the access whitelist
.exec help Displays all available commands in a formatted embed
